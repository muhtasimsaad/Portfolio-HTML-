Sudoku Solver

Sudoku is a puzzle based game that has a grid of 9 3x3 boxes, making it a grand total of a 9x9 grid. However this game has only one simple rule which is enough to make it as hard as it is. The rule is "No number can be repeated in any Row,Column and the 3x3 boxes."

This C# application can take any sudoku puzzle as input and solve it within 100-200 miliseconds using DFS algorithm. The different colors in the solution determines if that perticular cell was given, solved using simple logic or had to be guessed and rectified with the help of DFS algorithm.
